BalloonRider Game Introduction

A game that fly up to the game world while acquiring and consuming fuel. The player becomes a hot air balloon character in the game. In Game world, There are Blue / Yellow / Red block in the middle as it floats on a given fuel consumption and avoid these obstacles appropriately You need to push it up with the fuel you have acquired and climb to the end of the stage.
